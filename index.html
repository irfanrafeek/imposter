To enable real multiplayer, add a free Firebase project:

  1. Go to console.firebase.google.com
  2. Create a new project
  3. Build → Realtime Database → Create (test mode)
  4. Project settings → Add Web app → copy config
  5. Paste config into FIREBASE_CONFIG at the top of index.html

See README.md for full instructions and database security rules.
